古文の基礎

歴史的仮名遣い

次の言葉を現代仮名遣いに直しなさい。

やうやく

＜解答＞　ようやく
＜解説＞　y + au→y + ou

よはひ

＜解答＞　よわい
＜解説＞（語頭ではないので）
　　　　　は→わ、ひ→い

ゆゑ

＜解答＞　ゆえ
＜解説＞　ゑ→え

たまうやう

＜解答＞　たもうよう
＜解説＞　m + au→m + ou
　　　　　y + au→y + ou

にぐわつ

＜解答＞　にがつ
＜解説＞　ぐわ→が

いみじう

＜解答＞　いみじゅう
＜解説＞　j + iu→j + yu u

つねならむ（「む」は助動詞）

＜解答＞　つねならん
＜解説＞（助動詞なので）
　　　　む→ん

をと

＜解答＞　おと
＜解説＞　を→お

ぢうご

＜解答＞　じゅうご
＜解説＞　ぢ→じ
　　　　　じうご
　　　　　j + iu→j + yu u

ほとる

＜解答＞　ほとる
＜解説＞　語頭なので変化しない。

あふぎ けんでうす

＜解答＞　おうぎ けんじょうす
＜解説＞（語中なので）ふ→う
　　　　au→ou
　　　　d + eu→d + yo u

次の単語を現代語訳しなさい。

やうやう

＜解答＞
次第に
＜解説＞
覚えましょう。


次の文章を品詞分解しなさい。

あはれ今年の秋も往ぬめり

＜解答＞
あはれ / 今年 / の / 秋 / も / 往ぬ / めり
＜解説＞
基本は単語の切れ目で分ける。
「往ぬめり」は動詞「往ぬ」に助動詞が付いた形。

いと幼ければ、籠にいれて養ふ

＜解答＞
いと / 幼けれ / ば /、籠 / に / いれ / て / 養ふ
＜解説＞
「幼ければ」は、「幼けれ」という形に「ば」がついた形
現代語のように「いれて」を「いれ / て」と分けることにも注意


この先の問題は、①品詞分解したうえで、②現代語訳し、③色付きの文字があれば品詞を答えなさい。

いとうつくしうてゐたり

＜解答＞
①いと / うつくしう / て / ゐ / たり
②とてもかわいらしい様子で座っている。 ③うつくし＝小さくてかわいらしい
＜解説＞
①「ゐたり」は「ゐる」＋「たり」なので分かれる。
②特にない。
③現代と意味が違う点に注意。このような語を古今異義語という。

おはするにて、知りぬ

＜解答＞
①おはする / にて / しり / ぬ
②いらっしゃるので、知りました
③おはする＝いらっしゃる
＜解説＞
①特にない
②日本語が多少片言でも、基本は直訳。
③「おはする」は尊敬語。「いる」と訳すと減点。

子になりたまふべき人なめり

＜解答＞
①子 / に / なり / たまふ / べき / 人 / な / めり
②子になりなさるはずの人であるようです
＜解説＞
①「なめり」の「めり」は、「往ぬめり」と同じ。
②「なりなさる」という日本語に違和感を感じるかもしれないが、こう覚えよう。

（竹取物語において）
よきほどなる人になりぬれば

＜解答＞
①よき / ほど / なる / 人 / に / なり / ぬれ / ば
②結婚するのによい大きさの人に成長したので ＜解説＞
①「なりぬれば」は、「ば」が単体で存在すると考えると考えやすい。
②前の句とつながっているという前提が必須。

かたち、けうらなることこの世になく

＜解答＞
①かたち / けうらなる / こと / 世 / に / なく
②容貌が清らかで美しい様は世間に類がなく ＜解説＞
①「けうらなる」はこれで一語。分けないよう注意。
②現代語訳する際は、語順を変えずに書くことを意識する。

竹取物語

基礎知識

次の問題に答えなさい。

竹取物語の成立時期を答えよ。

＜解答＞
平安時代初期

「うつくし」のように、現代語と古語で用法が異なる語を何というか。

＜解答＞
古今異義語
＜解説＞
ちなみに「ここんいぎご」と読む。「こきん」ではない。

竹取物語の文体の特徴を答えよ。

＜解答＞
素朴で簡潔な文体
＜解説＞
同じ言葉を繰り返し使う場面などにその一面が現れている。

重点

次の問題に答えなさい。本文中から抜き出す際は、教科書を使用してもよい。

本文中で掛詞が使われている場所を抜き出しなさい。

＜解答例＞
「子」と「籠」
＜解説＞
どちらも「こ」と読む。

上の問のような、いわゆる「言葉遊び」を何というか答えなさい。

＜解答＞
言語遊戯
＜解説＞
掛詞だけでなく縁語も含まれる。

かぐや姫を大切に扱っているとわかる描写を、敬語を使っていない表現で一つ抜き出しなさい。

＜解答例＞
・手にうち入れて
・帳のうちよりも出ださず ＜解説＞
基本的に上のどちらかを選択して答える。

「翁やうやう豊かになりゆく」の要因となった部分を抜き出しなさい。

＜解答＞
よごとに黄金ある竹を見つくること重なりぬ
＜解説＞
前の「節を隔てて」が含まれてもよい。

髪あげ、裳着、着付けは、何の儀礼であるとともに、何の適齢期であることを表したか。

＜解答＞
成人儀礼、結婚適齢期
＜解説＞
当時の女性の成人の儀式が羅列されている。

和歌

基礎知識

次の（）に入る言葉を答えなさい。

平安時代初期まで、貴族男子の文学は（）のみだった。

＜解答＞
漢詩文
＜解説＞
和歌は女性のものだった。

天皇の命令によって編纂された（1）集である（2）が、男性が和歌をたしなむきっかけとなった。

＜解答＞
（1）：勅撰
（2）：『古今和歌集』 ＜解説＞
勅撰の字に注意。
